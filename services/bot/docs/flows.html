Потоки бота

Краткая визуальная шпаргалка по ключевым сценариям работы.

Покупка подписки баланс

  1. Пользователь: «Тарифы»
  2. Бот: карточка тарифа + кнопки
  3. Пользователь: tariff:connect
  4. Бот: выбор страны
  5. Пользователь: country:fi
  6. Бот: варианты оплаты
  7. Пользователь: pay:balance
  8. Бот: списание баланса + создание клиента XUI
  9. Бот: запись подписки в БД + Redis
  10. Бот: отправка инструкции и ссылки

Личный кабинет

  1. Пользователь: «Личный кабинет»
  2. Бот: запрос статуса подписки в XUI
  3. Бот: чтение подписки из БД/Redis
  4. Бот: вывод статуса + инструкции
  5. Если подписки нет — кнопка «Купить»

Реферальный старт

  1. Пользователь: /start <ref_tg_id>
  2. Бот: ensure_user
  3. Бот: set_referrer
  4. Бот: приветствие + меню
Обновляй при изменении сценариев в `services/bot/app/handlers/start.py`.
